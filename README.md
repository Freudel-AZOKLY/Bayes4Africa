# Bayes4Africa
 Ce Dashboard Veille Économique Streamlit suit les Énergies Renouvelables en Afrique Francophone, collecte les données via l'API Banque Mondiale et les analyse par régression bayésienne (PyMC). Il fournit prédictions, diagnostics et alertes email  si un indicateur dépasse un seuil critique.

This Economic Monitoring Dashboard in Streamlit tracks Renewable Energy in Francophone Africa, collecting data via the World Bank API and analyzing it with Bayesian regression (PyMC). It provides forecasts, diagnostics, and email alerts when indicators exceed critical thresholds.
